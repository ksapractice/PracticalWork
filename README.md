# PracticalWork

### Use Node.js
#### Install
* Install `Node.js v14.x`
* Run `npm i` to install dependencies

#### Create file
* `.env` (in main directory)
  * DB_HOST = `host database (MySql)`
  * DB_USER = `db username`
  * DB_PASS = `db password`
  * DB_NAME = `db name`
  * DB_TABLE_NAME = `db table name`
  * site = `webserver host name/ip`

#### Finish
* Run `app.js`



### Use Docker
* `docker-compose up db`
* `docker-compose up web`
