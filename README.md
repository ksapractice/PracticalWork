# getDataMysql
 
#### Install
* Install `Node.js`
* Run `npm i` to install dependencies

#### Create file
* `.env` (in main directory)
  * DB_HOST = `host database (MySql)`
  * DB_USER = `db username`
  * DB_PASS = `db password`
  * DB_NAME = `db name`
  * DB_TABLE_NAME = `db table name`
  * site = `webserver host name/id`

#### Finish
* Run `app.js`
